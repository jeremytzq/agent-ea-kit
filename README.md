# Real Estate Agent EA Kit

**Your AI executive assistant, ready in 60 minutes.**

A plug-and-play template that gives Singapore real estate agents a personal AI assistant — built on Claude Code, pre-loaded with skills for listings, social media, follow-ups, calendar, and research.

---

## What's Inside

| | |
|---|---|
| **5 Built-In Skills** | Listing writeups, social media content, WhatsApp follow-ups, calendar management, Singapore property research |
| **Context Files** | Pre-built files your assistant uses to understand your brand, goals, and business |
| **Setup Guide** | Step-by-step walkthrough — no coding required |
| **Decision Log** | Append-only log so your assistant gets smarter over time |
| **Templates** | Session summary and other reusable formats |

---

## Skills

| Skill | What It Does |
|---|---|
| `listing-writeups` | HDB, condo, and rental descriptions for PropertyGuru, 99.co, SRX, and Instagram |
| `social-media-content` | Instagram captions, hooks, and hashtags in your voice |
| `follow-up-templates` | WhatsApp messages for every deal stage |
| `calendar-management` | Daily briefings, viewing scheduling, follow-up reminders via Google Calendar |
| `research` | Deep Singapore property market research via Perplexity API |

## Getting Started

Read **[SETUP.md](./SETUP.md)** — it walks you through everything in 45–60 minutes.

The short version:
1. Clone or download this repo
2. Create a private GitHub repo and upload these files
3. Connect it to Claude Code at [claude.ai/code](https://claude.ai/code)
4. Fill in your details in the `context/` files
5. Type `Hi [ASSISTANT_NAME]` to start your first session
